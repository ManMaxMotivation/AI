# Manual Exploratory Checks

| ID | Scenario | Expected observation |
| --- | --- | --- |
| MANUAL-01 | Open reserved and unavailable item pages on desktop. | Label is understandable, visible, and does not disrupt layout. |
| MANUAL-02 | Use keyboard and pointer on disabled desktop, mobile, and compare actions. | No interaction opens checkout or creates a misleading focus path. |
| MANUAL-03 | Complete the available-item journey in the approved environment. | Existing user flow remains intact. |
| MANUAL-04 | Observe the reservation transition while page data refreshes. | A user cannot reach an avoidable error from an active stale action. |

Actual result, environment, evidence, and status must be recorded only when a
tester has performed the scenario.
